Personal Finance Tracker

Java kullanılarak geliştirilmiş console tabanlı kişisel finans takip uygulamasıdır.
Bu proje, kullanıcıların gelir ve giderlerini kaydetmesini, işlemleri listelemesini ve anlık bakiyesini hesaplamasını sağlar.

Projenin Amacı

Bu proje;

Java’da Nesne Tabanlı Programlama (OOP) yapısını uygulamak

ArrayList ile dinamik veri yönetimi yapmak

Switch-case menü sistemi kurmak

Temel finans hesaplamalarını gerçekleştirmek

Console uygulama mantığını pekiştirmek

amacıyla geliştirilmiştir.

 Özellikler

Gelir ekleme

Gider ekleme
